Routine Programming

This is my current routine. It follows a basic PPL (Pull Push Legs) split across multiple days of the week. This means that one day is devoted to exercises where I push weight away from myself, one day is for exercises where I pull weight towards myself, and one day is for leg and core exercises.

Weekday Split Exercises
Monday Push
  1. Overhead Press
  2. Incline Dumbbell Bench Press
  3. Close-Grip Barbell Bench Press
  4. Preacher Curl
  5. Dumbbell Raises
  6. Parallel Bar Dips
  7. Run 2 miles
Tuesday Legs
  1. Barbell Squat
  2. Zercher Squat
  3. Dumbbell Step Up
  4. Leg Press
  5. Standing Calf Raises
  6. Crunch Machine (Slow Reps)
  7. Keiser Bike for 5-10 Minutes
Wednesday Rest ----------
Thursday Pull
  1. Deadlift OR Hex Bar Deadlift
  2. Bent-Over Barbell Row
  3. Dumbbell Shrug
  4. OH Iso-Lateral Row
  5. Lat Pulldown OR Seated Cable Pull
  6. Floor Face Pull
  7. Rowing Machine 2000 Meters
Friday Push
  1. Flat Barbell Bench Press
  2. Incline Barbell Bench Press
  3. Seated Overhead Dumbbell Press
  4. Dumbbell Raises
  5. Preacher Curl
  6. Parallel Bar Dips
  7. Rowing Machine 2000 Meters
Saturday Legs
  1. Barbell Squat
  2. Zercher Squat
  3. Leg Press
  4. Dumbbell Step Ups
  5. Multi-Plane Calf Machine
  6. Weighted Crunches
  7. Keiser Bike 5-10 Minutes
Sunday Pull
  1. Hex Bar Deadlift
  2. Barbell Row
  3. Iso-Lateral Row (OH)
  4. Iso-Lateral Front Lat Pulldown
  5. Dumbbell Shrug
  6. Floor Face Pulls
  7. Rowing Machine 2000 Meters
